# + Beleza
Website for presenting a beauty salon with tarot

```bash
Using {
  Website:
    - Next.js {
      React
      TypeScript
      Tailwind
      pnpm
      Sass
      shadcn/ui
    }
}
```

## How to start the application

### Run front-end
```bash
npx pnpm run dev

```

### or

```bash
npx pnpm run build
npx pnpm run start

```
